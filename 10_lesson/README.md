# Проект автоматизации тестирования - Lesson 10

Проект содержит автоматизированные тесты для веб-приложений с использованием Selenium, Pytest и Allure Framework.

## Технологии

- **Python 3.7+** - язык программирования
- **Selenium 4+** - автоматизация браузера
- **Pytest** - фреймворк для тестирования
- **Allure Framework** - генерация отчетов
- **Page Object Pattern** - паттерн проектирования
- **WebDriver Manager** - автоматическое управление драйверами
